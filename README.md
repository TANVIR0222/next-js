# Next.js App (workspace)

This repository is a Next.js app (App Router) with a simple About page. Use this README to run, build and explore the project.

## Getting started

Prerequisites:

- Node.js (16+)
- npm, yarn, pnpm or bun

Install dependencies:

npm run dev

# or

yarn dev

# or

pnpm dev

# or

bun dev

## Notes

The About component is implemented as About.
Open page.tsx to edit it.
Use the scripts in package.json for linting, testing and building.
The
.next folder contains build artifacts and should remain ignored by Git.
